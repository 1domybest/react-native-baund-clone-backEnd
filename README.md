## [Baund]바운드 클론 코딩

# 기능

- [구글로 로그인]
- [JWT 에 로그인정보 저장후 AsyncStorage 를 활용해 저장후 사용]
- [AWS S3 를 이용한 프로필 사진 업로드 기능]
- [임시비밀번호, 이메일 인증번호 spring boot nodeMailer 를 사용해 구현]
